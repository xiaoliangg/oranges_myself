2.1.4　用Bochs调试操作系统
1、直接在bochs的安装目录中双击bochsdbg文件，然后选择加装前面我们的配置文件bochsrc-sample.bxrc，点开始就进入调试模式
2、注意查看 a.img 是否写入成功，可以使用16进制软件查看
3、help查看相关指令 如: help;help info;    有的命令修改了(如dump_cpu)，记得查看help
